# 🔐 Password Encryption using Matrix Multiplication

## 📘 Project Description
This project demonstrates encryption and decryption using Linear Algebra concepts. It uses matrix multiplication to encrypt a password and inverse matrix to decrypt it.

## 🧠 Algorithm Used
Hill Cipher (Matrix-based encryption)

## ⚙️ Technologies Used
- Python
- NumPy

## 🚀 How to Run
1. Open terminal
2. Go to project folder
3. Run:
   python matrix_project.py

## 📊 Features
- Encrypt password
- Decrypt password
- Uses matrix multiplication
- ASCII conversion
